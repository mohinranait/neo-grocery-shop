## NEO GROCERY STORE

### Techonology

- Mongoose.js
- Express.js,Node.js
- MongoDB
- JWT authentication
- Verify email process using Nodemailer

### Features

- Role management
- User management
  - Register with email verification
  - Login
  - Forgot Password
  - Reset Password
  - User Active, Inactive, Band
  - Update Profile
- Brand CRUD
- Product attribute and attribute configuration
- Categories CRUD
- Product store full controll by admin and manager
